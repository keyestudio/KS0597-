# 6 问题解答

1. kidsblock始终连接不上？

   - 请确保没有其他程序占用了通信串口。若Arduino IDE未完全关闭，可能导致串口资源被占用。

2. Arduino IDE 上传代码时出现了 "uploading error: exit status 2" 的错误？

   - 串口占用问题：请确保没有其他程序占用通信串口。

   - 硬件问题：检查与主板连接的电缆和接线是否正确连接，并确保主板正常工作。
